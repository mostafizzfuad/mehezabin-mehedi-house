A fully responsive, modern e-commerce storefront built with **Next.js 16**, **Tailwind CSS**, , **ShadCN UI** and **TypeScript**. This project demonstrates advanced frontend development techniques, including robust state management, complex form validation, and interactive UI design.

## 🚀 Features

Based on the implemented modules, this project includes:

- **Responsive Modern UI:** Fully responsive Layout, Navbar, Footer, and Homepage designed with Tailwind CSS & ShadCN UI.
- **Interactive Product Cards:** Dynamic product cards with the ability to change product images based on the selected product color.
- **Advanced Filtering & Routing:** Utilizes Next.js URL Search Params for efficient e-commerce product filtering.
- **Robust Cart State Management:** Implements **Zustand** for global cart state, including actions (add/remove) and data persistence using Local Storage (with a fix for Next.js hydration issues).
- **Multi-Step Checkout & Validation:** Features a comprehensive cart page with steps, utilizing **React-Hook-Form** and **Zod** for strict payment form validation.
- **Dynamic Pages:** Detailed Single Product & Category Page and Product List Page designs.
- **SEO Optimized:** Implemented Next.js Metadata for better search engine visibility.

## 🛠️ Tech Stack

- **Framework:** Next.js 16
- **Language:** TypeScript
- **Styling:** Tailwind CSS, ShadCN UI
- **State Management:** Zustand
- **Form Handling & Validation:** React-Hook-Form, Zod

## ⚙️ Setup & Installation

Follow these steps to run the project locally on your machine:

### Clone the Repository
```bash
git clone https://github.com/mostafizzfuad/mehezabin-mehedi-house.git
cd mehezabin-mehedi-house
npm install
```
